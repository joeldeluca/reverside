Prototype jouable (web)

REVERside 3D

Puzzle cylindrique à rotations par niveaux + pivot central. Objectif : reconstituer l’alignement cible (recto/verso selon le défi).

Jouer Voir l’aide
Astuce : si une mise à jour ne s’affiche pas, rafraîchissez la page (ou navigation privée).

Raccourcis de base

  • Rotations : boutons NH / N4 / NM / N2 / NB.
  • Pivot : bouton P (colonne H).
  • Colonne : Col ↑ / Col ↓.
  • Images : « Charger recto » / « Charger verso » pour personnaliser.

Note : iOS 12 et plus ancien n’est pas ciblé pour l’instant. Objectif actuel : Safari iOS 15+ et desktop.
